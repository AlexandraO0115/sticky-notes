Sticky Notes Application

A simple and intuitive Sticky Notes application that allows users to create, edit, and delete notes. This project demonstrates core programming concepts, clean project structure, and version control using Git and GitHub.

Features:

- Create new sticky notes
- Edit existing notes
- Delete notes
- Simple and user-friendly interface

Purpose of This Project:

This project was created as part of my developer portfolio to demonstrate application development fundamentals, clean code organization, and the use of Git and GitHub.
